# Mehadi Hasan Reaz - Portfolio

A modern, responsive portfolio website built with React, TypeScript, and Tailwind CSS.

## Features

- Responsive design
- Dark mode by default
- Project showcase with links to live apps
- Skills and experience sections
- Contact form
- Smooth animations and transitions

## Technologies Used

- Vite
- TypeScript
- React
- shadcn/ui
- Tailwind CSS
- React Router DOM

## Getting Started

1. Clone the repository:
```sh
git clone https://github.com/MehadiReaz/reaz-flutter-profile.git
cd reaz-flutter-profile
```

2. Install dependencies:
```sh
npm install
```

3. Start the development server:
```sh
npm run dev
```

4. Build for production:
```sh
npm run build
```

## Deployment

The site is automatically deployed to GitHub Pages when changes are pushed to the main branch.

Visit the live site at: https://mehadireaz.github.io/reaz-flutter-profile/
